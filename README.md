# Análise Lógica de Hábitos Diários

## Objetivo
Analisar hábitos diários utilizando lógica proposicional e estatística básica, buscando identificar padrões de produtividade.

## Hipótese
A produtividade tende a aumentar quando o tempo de sono é maior ou igual a 7 horas e o tempo de estudo é maior ou igual a 3 horas.

## Metodologia
Os dados foram coletados de forma manual e analisados por meio de regras lógicas do tipo SE/ENTÃO, além de cálculos simples como médias e comparações.

## Regras Lógicas
- Se sono < 7, então produtividade baixa
- Se sono ≥ 7 e estudo ≥ 3, então produtividade alta
